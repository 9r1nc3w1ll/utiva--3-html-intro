Images in HTML

Basic Image Syntax

<img src="image.jpg" alt="Description of image" />

  • src - Specifies the path to the image
  • alt - Provides text description (for accessibility and if image fails to load)

Image Formats

Format Best For Features
JPEG (.jpg, .jpeg) Photos and complex images
  • Millions of colors
  • Lossy compression
  • No transparency
PNG (.png) Images with transparency, logos, text
  • Lossless compression
  • Supports transparency
  • Good for screenshots and graphics
GIF (.gif) Simple animations, icons
  • Limited to 256 colors
  • Supports animation
  • Supports transparency (binary only)
SVG (.svg) Logos, icons, illustrations
  • Scalable Vector Graphics
  • Maintains quality at any size
  • Text-based format (can be edited with text editor)
WebP (.webp) Web images, modern browsers
  • Better compression than JPEG and PNG
  • Supports both lossy and lossless compression
  • Supports animation and transparency
