Links in HTML

Basic Link Structure

Link Text

The a stands for "anchor" and the href attribute specifies the destination.

Types of Links

Relative Links (within your site)

  • About Us - Links to a file in the same directory
  • View Product - Links to a file in a subdirectory
  • Home - Links to a file in the parent directory

Absolute Links (to other websites)

  • Visit Example.com - External website
  • MDN Web Docs - Mozilla Developer Network

Link Targets

  • Example.com (new tab) - Opens in a new tab/window
  • Same frame - Default behavior
  • Parent frame - For use with frames/iframes
  • Full window - For use with frames/iframes

Email Links

  • Send Email - Basic email link
  • Contact with Pre-filled Subject and Body

Media Links

  • Download PDF
  • View Image
  • Download ZIP

Links to Page Sections

  • Go to Section 1 - Links to a section on the same page
  • Go to Section 2 on another page - Links to a section on another page

Additional Link Attributes

  • Hover over me - Shows a tooltip when hovering
  • Example with rel attribute - Tells search engines not to follow this link
  • Download this page - Suggests downloading rather than navigating
